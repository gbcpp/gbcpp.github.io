# 更新日志

## 大事记

### 2021

- 2021年12月22日，发布 [H20-ac](https://github.com/zhonger/jekyll-theme-H20-ac) Jekyll 主题的第一个公开版本。
- 2021年12月24日，正式支持 gem 方式使用本主题。
- 2022年1月6~11日，修复若干错误及优化了某些小细节。
- 2022年4月29日~30日，新增若干功能，修复某些小细节
  - 新增深色模式切换按钮
  - 新增全站一键灰度化
  - 新增时间格式配置
  - 新增跟随内容索引导航高亮

  - 修复了深色模式下一些有问题的样式。
