Ubuntu Postfix SMTP 邮件通知

Ubuntu 搭建 Postfix 邮件 SMTP 发送服务器

zhonger Mon, 25 Jun 2018 11:30:00 +0900
本文总共 354 字 · 阅读全文大约需要 2 分钟

从零开始几步搭建好自己邮件通知服务器

sudo apt install mailutils -y
sudo vim /etc/postfix/main.cf

# /etc/postfix/main.cf
myhostname = {domain}
mydestination = {domain}, $hostname, localhost.localdomain, localhost
net_interfaces = loopback-only

sudo hostnamectl set-hostname {domain}

sudo vim /etc/mailname
# /etc/mailname
{domain}

sudo service postfix restart

echo "This is the body of email" | mail -s "This is the subject line" <收件人地址>
